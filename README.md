# Student Management System

##  Project Description

This is a menu-driven Student Management System developed in Python. It allows users to manage student records using file-based storage (JSON).

##  Features

- Add Student
- View Student Records
- Search Student
- Update Student Details
- Delete Student
- Duplicate Roll Number Check
- Input Validation
- JSON File Storage

##  Technologies Used

- Python 3
- JSON
- VS Code

##  Project Structure

Student-Management-System/
│── main.py
│── students.json
│── README.md

##  How to Run

1. Install Python 3.
2. Open the project in VS Code.
3. Run:

```bash
python main.py
```
